# jsp-blog
jsp+servlet实现的个人博客系统

演示站点：https://nichang.site/

### 技术栈
`Jsp/Servlet` `Jquery`  `Bootstrap` `Mysql` `Ajax`  `Markdown`  `响应式`

### 已有功能
###### 前台：
查看文章列表
查看文章详情
关于我
工程
联系我

###### 后台：
登录（验证码）
写文章（markdown）
修改文章
删除文章
对分类进行增删改查
两层分类
